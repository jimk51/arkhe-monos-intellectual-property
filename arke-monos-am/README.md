# ΑΡΚΗ ΜΟΝΟΣ – AM

## Position dans le système ARKÈ MONOS

AM est une **variante de signature**, transversale à l’ensemble du système.

Elle peut représenter :
- l’ensemble d’ARKÈ MONOS
- ou accompagner chaque variante principale
  (ΑΞΙΣ, ΟΡΙΓΟ, ΝΕΞΥΣ, ΑΣΚΕΝΣΙΟ, ΚΟΡΕ, ΜΟΝΑΔΑ, ΑΠΕΞ, ΓΕΝΕΣΙΣ)
